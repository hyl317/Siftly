# DaVinci Resolve Keyboard Shortcuts (macOS)

Exported from DaVinci Resolve Studio 20. On macOS: Ctrl = Cmd, Alt = Option.

Shortcuts listed as `Cmd+Key` in this file correspond to the physical Cmd key on Mac keyboards.
Unassigned shortcuts (no key listed) are omitted.
